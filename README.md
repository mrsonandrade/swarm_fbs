# Wave Simulation App

This interactive app simulates the response of 10 rectangles in a 2D wave field using SciPy's `odeint`.

## Features

- User input: wave direction (180° to 225°) and period (0.79s to 11.07s)
- Simulation and animation of wave-induced motion
- Interactive interface using Streamlit

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
